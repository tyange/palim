Palim은 **한국 시를 한 글자도 안 틀리게 따라 칠 수 있는** 시 전사 에디터입니다.
한국 시는 배치 자체가 시인 부류와 자연스럽게 흐르는 산문 부류로 나뉘어, **두 가지 쓰기
모드**를 제공합니다. 어느 쪽이든 화면 그대로 **SVG/PNG 이미지**로 저장할 수 있습니다.

## 🎯 주요 기능

- **격자(grid) 모드** — 활자의 2D 위치가 곧 시인 **배치형 시**용
  - 고정폭 격자에 자소 1개 = 1칸으로 놓아 위치를 결정적으로 보존(공백도 그대로)
  - 칸 클릭·방향키로 어디든 직접 배치, 격자는 내용에 맞춰 자동 확장
- **흐름(flow) 모드** — 보통 공백 + 자연스러운 줄 흐름의 **산문·일반 시**용 비례폭 조판
- 각 모드는 **편집 화면과 출력이 같은 모습**(WYSIWYG)
- **한국어 IME 입력** — 조합 중인 글자도 실시간 반영. 빈 칸·행두 클릭 시 캐럿 삽입 위치를 정확히 보정 (자세한 내용은 [docs/korean-ime-bug.md](docs/korean-ime-bug.md))
- **이미지 내보내기** — 화면 그대로 **SVG** 저장, 웹폰트를 픽셀에 구워낸 고해상도 **PNG** 저장
