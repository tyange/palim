Palim은 한국어 **원고지**를 웹으로 옮긴 텍스트 에디터입니다. 글을 입력하면
원고지 칸 점유 규칙에 맞춰 격자에 자동으로 흘러 들어가고, 그대로 **SVG/PNG 이미지**로
저장할 수 있습니다.

## 🎯 주요 기능

- **원고지 편집 방식을 그대로 채용**
  - 한글은 한 칸, 숫자·소문자는 두 글자씩 한 칸, 문장부호+따옴표는 한 칸에 합치는 등 원고지 점유 규칙을 그대로 구현
  - 줄 첫 칸에 올 수 없는 구두점(`. , ? !`, 닫는 따옴표)을 앞 줄 오른쪽 여백으로 자동 배치
- **한국어 IME 입력** — 조합 중인 글자도 실시간 반영. 빈 칸·행두 클릭 시 캐럿 삽입 위치를 정확히 보정 (자세한 내용은 [docs/korean-ime-bug.md](docs/korean-ime-bug.md))
- **이미지 내보내기** — 화면 그대로 **SVG** 저장, 웹폰트를 픽셀에 구워낸 고해상도 **PNG** 저장
